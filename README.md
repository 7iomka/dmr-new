# How to run

1. Open terminal and run `php -S localhost:8000`
2. Open url `localhost:8000/dashboard.php` in your browser.


## UI Rules (IMPORTANT)

### See: docs/ui-guidelines

- Cards → rounded-xl ONLY
- Buttons/inputs → rounded-lg
- Small elements → rounded-md
- Do not use random Tailwind radius classes
- Prefer semantic classes: card, btn, input
